Sistema de Gestão de Tarefas - Documentação Step by Step

Objetivo:
Desenvolver uma aplicação em Python que permita gerir tarefas pessoais (um To-Do List),
de forma simples e organizada.

Funcionalidades Mínimas:
1. Adicionar uma nova tarefa
2. Listar tarefas
3. Concluir uma tarefa
4. Remover uma tarefa
5. Sair do programa

Descrição do Código:

1. Importação de bibliotecas:
   - json: para salvar e carregar a lista de tarefas em um arquivo JSON.
   - datetime: para registrar a data e hora de criação e conclusão das tarefas.

2. Estrutura de dados:
   - tarefas: lista que armazenará dicionários, cada um representando uma tarefa.
   - Cada tarefa contém:
     - descricao: texto da tarefa
     - concluida: True/False
     - criada_em: data e hora em que foi criada
     - concluida_em: None inicialmente, atualizado quando a tarefa é concluída

3. Funções:

   - carregar():
     - Carrega o conteúdo do arquivo 'memoria_tarefas.json' para a lista tarefas.
     - Se o arquivo não existir, inicia a lista vazia.

   - salvar():
     - Salva a lista tarefas no arquivo 'memoria_tarefas.json'.
     - Só é chamado quando o usuário escolhe sair do programa.

   - mostrar_menu():
     - Exibe as opções do sistema para o usuário.

   - adicionar_tarefa():
     - Solicita ao usuário o nome da tarefa.
     - Cria um dicionário com a tarefa, data/hora atual e campo concluida_em como None.
     - Adiciona o dicionário à lista tarefas.

   - listar_tarefas():
     - Verifica se há tarefas registradas.
     - Para cada tarefa, mostra:
       - número
       - descrição
       - status (Concluída/Pendente)
       - data de criação
       - data de conclusão, se houver

   - concluir_tarefa():
     - Mostra a lista de tarefas.
     - Solicita o número da tarefa a ser concluída.
     - Atualiza o campo 'concluida' para True.
     - Atualiza 'concluida_em' com a data/hora atual.
     - Pergunta se deseja remover a tarefa concluída da lista.

   - remover_tarefa():
     - Mostra a lista de tarefas.
     - Solicita o número da tarefa a ser removida.
     - Remove a tarefa da lista usando pop().

4. Programa principal:
   - Chama carregar() ao iniciar.
   - Loop infinito mostrando o menu e tratando opções:
     1: adicionar_tarefa()
     2: listar_tarefas()
     3: concluir_tarefa()
     4: remover_tarefa()
     5: salvar() e sair
   - Validação de opção inválida.

Observações:
- As datas são armazenadas em formato 'dd/mm/yyyy HH:MM:SS'.
- O campo 'concluida_em' inicia como None, indicando que a tarefa ainda não foi concluída.
- Ao concluir, o campo recebe a hora real de conclusão.

Funcionalidades Extras:
- Registro de data e hora de criação e conclusão de tarefas.
- Possibilidade de remover automaticamente tarefas concluídas.

Instruções de utilização:
1. Certifique-se de ter Python instalado.
2. Salve o código em um arquivo .py (por exemplo, tarefas.py).
3. Execute no terminal: python tarefas.py
4. Siga as instruções no menu.
5. O arquivo 'memoria_tarefas.json' será criado para armazenar as tarefas.
